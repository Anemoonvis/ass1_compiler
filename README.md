# CoCo

This gives a base for your own civicc compiler.
Clone this repo: 
```bash
git clone git@github.com:CoCoNut-UvA/civicc-skeleton.git civicc
cd civicc
./configure.sh
```
If everything went correctly, a build-debug directory is created. 
Run
```bash
make -C build-debug
```
to build your compiler.
